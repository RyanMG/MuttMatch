# Mutt Match
Find your next favorite pup!
_Coding sample for Fetch_

Built with NextJS. Deployed on Vercel

## Local testing
Install components
```
* npm install
or
* pnpm install
```
You can start the application with:
```
* pnpm start
or
* npm run start
```
Then navigate to `http://localhost:3000`
